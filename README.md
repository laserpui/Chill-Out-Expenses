# ⛺ Chill Out Expense Tracker

เว็บแอปพลิเคชันบันทึกค่าใช้จ่ายการเดินทางท่องเที่ยวสำหรับกลุ่ม **"Chill Out"** ด้วยดีไซน์สไตล์มินิมอลธรรมชาติ (Sage Green Tone) รองรับการแสดงผลเรียลไทม์แดชบอร์ด อัปโหลดรูปภาพใบเสร็จขึ้น Google Drive และจัดเก็บข้อมูลลง Google Sheets โดยตรง ปราศจากปัญหา CORS บล็อก ทำงานโฮสต์ได้อย่างสมบูรณ์แบบ 100% 

---

## ✨ คุณสมบัติของระบบ (Key Features)
* **บันทึกค่าใช้จ่ายอย่างราบรื่น:** รองรับหัวข้อวันที่จ่าย, ประเภทค่าใช้จ่าย, รายละเอียดเพิ่มเติม, และจำนวนเงิน
* **ฟังก์ชันแนบรูปใบเสร็จอัจฉริยะ:** ถ่ายรูปหรือเลือกภาพจากเครื่อง บีบอัดขนาดไฟล์ภาพด้วย HTML5 Canvas โดยอัตโนมัติก่อนอัปโหลด เพื่อประหยัดพื้นที่และโหลดเร็วขึ้น
* **ตัวเลือก "คนอื่น ๆ (ระบุชื่อเอง)":** หากมีสมาชิกพิเศษนอกระบบ สามารถกดระบุชื่อเองได้ และรายชื่อนั้นจะปรากฏขึ้นบนแดชบอร์ดสรุปยอดจ่ายเงินโดยอัตโนมัติ
* **Real-time Dashboard:** แสดงผลสรุปยอดใช้จ่ายรวมทั้งทริปสะสม, สัดส่วนการจ่ายเงินรายคน, สัดส่วนประเภทค่าใช้จ่าย และรายการประวัติย้อนหลัง 10 รายการเรียลไทม์
* **สลับหน้าการใช้งานลื่นไหล (SPA):** เปลี่ยนหน้าบันทึกและหน้ารายงานได้อย่างฉับไวโดยไม่ต้องโหลดหน้าเว็บใหม่
* **Responsive Design:** ออกแบบอย่างใส่ใจเพื่อการใช้งานผ่านเบราว์เซอร์บนสมาร์ทโฟนและแท็บเล็ตได้อย่างดีเยี่ยม ไร้ปัญหาขอบจอหรือกล่องแจ้งเตือนเกยทับเมนู

---

## 📂 โครงสร้างไฟล์ในโฟลเดอร์นี้
* **`index.html`**: หน้ากากและโครงสร้างหลักของเว็บไซต์
* **`style.css`**: การตกแต่งสไตล์ธรรมชาติ Sage Green และเอฟเฟกต์การเคลื่อนไหวต่างๆ
* **`app.js`**: ตัวควบคุม Logic การทำงานฝั่งหน้าบ้าน (Frontend)
* **`google-apps-script.js`**: สคริปต์หลังบ้านสำหรับติดตั้งใน Google Apps Script (`Code.gs`) ทำหน้าที่ควบคุม Google Sheets & Google Drive
* **`index-google-apps-script.html`**: ไฟล์เวอร์ชันรวมไฟล์เดียว (Inline CSS/JS) สำหรับคัดลอกไปทำหน้าเว็บใน Google Apps Script โดยตรง (กรณีโฮสต์บน Apps Script ทั้งหมด)

## 🚀 ขั้นตอนการติดตั้งและเปิดใช้งานผ่าน GitHub Pages

คุณสามารถนำไฟล์หน้าบ้านแยกชิ้น (**`index.html`**, **`style.css`**, และ **`app.js`**) ไปโฮสต์บน **GitHub Pages** เพื่อให้ได้หน้าเว็บส่วนตัวที่มีความรวดเร็ว จัดการง่าย และเป็นทางการยิ่งขึ้นตามขั้นตอนดังนี้ครับ:

### ขั้นตอนที่ 1: ตั้งค่าหลังบ้านบน Google Apps Script
1. นำโค้ดในไฟล์ `google-apps-script.js` ไปติดตั้งบน [Google Apps Script](https://script.google.com) 
2. ตั้งค่า **Deploy as Web App** (เรียกใช้เป็น: **ฉัน**, สิทธิ์การเข้าถึง: **ทุกคน**)
3. คัดลอกลิงก์ Web App (ลิงก์ที่ลงท้ายด้วย `/exec`) ที่ได้มา

### ขั้นตอนที่ 2: ฝังลิงก์เชื่อมต่อในโค้ดฝั่งหน้าบ้าน
1. ดับเบิ้ลคลิกเปิดไฟล์ **`app.js`**
2. นำลิงก์ Web App ที่ได้จากขั้นตอนที่ 1 มาวางใส่ในตัวแปร `APPS_SCRIPT_URL` ที่บรรทัดแรก:
   ```javascript
   // บรรทัดที่ 2 ในไฟล์ app.js
   const APPS_SCRIPT_URL = "https://script.google.com/macros/s/xxxxxxxxx/exec";
   ```
3. บันทึกและปิดไฟล์

### ขั้นตอนที่ 3: อัปโหลดขึ้น GitHub และเปิดใช้งาน Pages
1. สมัครใช้งานหรือเข้าสู่ระบบที่ [GitHub](https://github.com)
2. คลิกสร้าง Repository ใหม่ตั้งชื่อโปรเจกต์ เช่น `chill-out-expense` (เลือกเป็น **Public**)
3. อัปโหลดไฟล์ **`index.html`**, **`style.css`**, และ **`app.js`** ขึ้นไปที่ Repository
4. ไปที่เมนู **Settings** (การตั้งค่า) ของ Repository -> เลือกหัวข้อ **Pages** ที่เมนูด้านซ้าย
5. ตรงหัวข้อ Build and deployment ให้เลือก Branch เป็น **`main`** (หรือ `master`) และโฟลเดอร์เป็น **`/ (root)`** -> คลิกปุ่ม **Save**
6. รอประมาณ 1-2 นาที ระบบจะสร้างลิงก์เว็บไซต์อย่างเป็นทางการให้คุณ เช่น `https://username.github.io/chill-out-expense/`

คุณและกลุ่ม Chill Out สามารถเปิดใช้บันทึกค่าใช้จ่ายผ่านลิงก์นี้จากมือถือของทุกคนได้ทันที! 🎉
